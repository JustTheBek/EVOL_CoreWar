Corewar Evolver


Tools and Programs

CCAI
A corewar evolver which successfully produced the first evolved warriors to enter the '94nop hill written by Barkley Vowk

Corewar Tribes
A Corewar evolver written by Linus Thorsell

CRE - Corewars Redcode Evolver
It is an open-source program that automatically evolves corewar warrios by using Charles Darwin's theories on natural selection. The programm is written by Loren Jamison

Fizzle
A corewar evolver written by Terry Newton.

ga_war.c
ga_war.c is a program that uses a genetic algorithm (GA) to evolve corewar warriors. The program is written by Jason Boer

Maezumo
Maezumo is a corewar evolver written by Christian Schmidt. Beside pure evolving it provides additionally hint modes to selectively evolve stone, scanner and paper.

µGP³
An evolutionary approach for generating Turing-complete corewar warriors. The program is written at the Politecnico di Torino, Dipartimento di Automatica e Informatica. For more details, see also the µGP³ Homepage

REBS - Redcode Evolver Bash Script
It is a corewar evolver bash script for Linux written by Terry Newton.

Redmaker
A corewar evolver written by Terry Newton.

RedMixer
This is Terry Newton's latest evolver.

RedRace
RedRace is a program that uses a genetic algorithm (GA) to evolve corewar warriors. It generated h1_36.red - the first evolved warrior to win a tournament. The program is written by David Hillis

Species
A corewar evolver written by Will 'Varfar'

SYS4
It is a corewar evolver using a set of perl scripts written by George Lebl. It simulates a two-dimensional array of redcode where the warriors battle their neighbors.

TEV - The Corewar Tiny Evolver
A corewar evolver with the absolute minimum code to make warriors capable of getting on the nano hill Written by John Metcalf

YACE - Yet Another Corewar Evolver
It is a modified version of ga_war.c written by Martin Ankerl
