Core War: pMARS

pMARS is a portable MARS written in C by Albert Ma, Nándor Sieben, Stefan Strack, and Mintardjo Wangsaw. The latest version supports the extended CWS'94 Redcode standard, p-space, read/write limits, and is the official Redcode simulator of rec.games.corewar. The first version of pMARS was released on 25th August 1993.

Current Development Snapshot

pMARS is actively maintained, and contributions are welcome. Please submit patches for review by email to John Metcalf or via the #pmars-dev Discord channel. We endeavour to review and apply patches to the dev snapshot within a couple of days. See the change log for a list of modifications since the last release.

pMARS-SDL

pMARS-SDL is a port of pMARS by Joonas Pihlaja, which uses the Simple DirectMedia Library to display the core. For more details, see the pMARS-SDL homepage. Pavel Šavara updated pMARS-SDL to create pMARS 0.9.3-a.
